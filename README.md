# Factorilab - Luxury Marketing Homepage

A ultra-modern, luxury marketing homepage for Factorilab - a studio that builds high-end websites, webapps, AI automations, AI infrastructure, and AI agents.

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **shadcn/ui**
- **Framer Motion** (for animations)
- **next-themes** (for theme management)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `/app` - Next.js App Router pages and layout
- `/components/ui` - shadcn/ui base components
- `/components/blocks` - Page sections and blocks
- `/lib` - Utility functions

## Design System

The design follows a luxury, minimal aesthetic:
- Deep charcoal/near-black backgrounds (`#050509`, `#0B0B10`)
- Soft neutral grays
- Subtle warm metallic accents (champagne gold)
- Clean typography with lots of negative space
- Smooth micro-animations

## Build

```bash
npm run build
```
